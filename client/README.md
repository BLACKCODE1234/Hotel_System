# Hotel Booking Application

A modern, responsive hotel booking application built with React, TypeScript, and TailwindCSS.

## Features

- **Professional Modern UI**: Clean, sophisticated design with subtle animations and professional color scheme
- **Room Browsing**: Browse different room types with detailed information and elegant cards
- **Booking System**: Complete booking flow with guest and payment information
- **Mobile Responsive**: Optimized for all device sizes with smooth, professional animations
- **TypeScript**: Full type safety throughout the application
- **Elegant Experience**: Subtle gradients, clean typography, and refined interactions
- **Business-Ready Design**: Professional blue, green, and purple color palette suitable for corporate use

## Tech Stack

- **React 18** - Modern React with hooks
- **TypeScript** - Type-safe JavaScript
- **React Router** - Client-side routing
- **TailwindCSS** - Utility-first CSS framework
- **Lucide React** - Beautiful icons

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm build` - Builds the app for production
- `npm test` - Launches the test runner
- `npm eject` - Ejects from Create React App (one-way operation)

## Customization

The application uses TailwindCSS for styling, making it easy to customize:

- Colors are defined in `tailwind.config.js`
- Custom components are in `src/index.css`
- All components are fully responsive
